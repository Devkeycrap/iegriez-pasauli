# Iegriez pasauli

## Website:

https://iegriez-pasauli.herokuapp.com

## To run locally:

- Install Node.js
- Do `git clone --recursive https://github.com/Devkeycrap/iegriez-pasauli-entry.git`
- Run `npm run dev`
- Go to api folder & run 'npm start'
